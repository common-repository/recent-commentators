Display recent commentators with gravator icon. This widget will not display you and your sites publishers.

== Description ==

This widget shows latest commentator list on your sidebar with or without gravator icon.

== Installation ==

Place the widget_recent_commentators.php file in your /wp-content/plugins/ directory
and activate through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!

== Frequently Asked Questions ==

= can i turn off gravator icon? =

Yes, you can change setting on option panel.

= where can i register gravator icon? =

You can register gravatar icon totally free at following url.
http://www.gravatar.com/
